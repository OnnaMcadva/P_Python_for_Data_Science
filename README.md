## ğŸš€ Modules

### Module 0: Starting
**Focus**: Python Basics & Fundamentals

Introduction to Python syntax, variables, data types, control structures, and basic programming concepts.  This module lays the groundwork for all subsequent learning.

**Topics Covered**:
- Variables and data types
- Control flow (if/else, loops)
- Functions and scope
- String manipulation
- Basic I/O operations
- Error handling

---

### Module 1: Array
**Focus**: NumPy & Array Operations

Deep dive into numerical computing with NumPy, including array creation, manipulation, indexing, slicing, and vectorized operations. 

**Topics Covered**: 
- NumPy array creation and initialization
- Array indexing and slicing
- Mathematical operations on arrays
- Broadcasting and vectorization
- Array reshaping and manipulation
- Performance optimization with arrays

---

### Module 2: DataTable
**Focus**: Pandas & Data Structures

Working with structured data using Pandas DataFrames, including data loading, cleaning, transformation, and basic analysis.

**Topics Covered**:
- DataFrame creation and manipulation
- Data loading from various sources
- Data cleaning and preprocessing
- Filtering and selection
- Grouping and aggregation
- Basic data analysis techniques

---

### Module 3: OOP
**Focus**: Object-Oriented Programming

Understanding and applying OOP principles in Python, including classes, inheritance, encapsulation, and polymorphism.

**Topics Covered**:
- Classes and objects
- Instance and class methods
- Inheritance and composition
- Magic methods (dunder methods)
- Encapsulation and data hiding
- Polymorphism and abstraction

---

### Module 4: DOD
**Focus**: Data-Oriented Design

Advanced design patterns and techniques for efficient data processing and manipulation in data science contexts.

**Topics Covered**: 
- Data-oriented design principles
- Performance optimization
- Memory management
- Efficient data structures
- Pipeline design patterns
- Best practices for data science code
